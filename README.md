# Система учёта складских запасов

Веб-приложение для учёта товаров на складе. Позволяет добавлять, редактировать и удалять товары, а также отслеживать их количество и стоимость.

## Работающее приложение

Ссылка на работающее приложение: https://mariyaprozzz.pythonanywhere.com

## Основные возможности

- Добавление товара (название, количество, цена)
- Редактирование количества и цены
- Удаление товара
- Автоматический подсчёт общей стоимости склада
- Подсветка товаров с низким остатком (меньше 10 штук)
- Предупреждение о товарах, которые заканчиваются

## Технологический стек

- Python + Flask (серверная часть)
- SQLite (база данных)
- HTML + CSS (интерфейс)

## Запуск проекта

Приложение уже запущено и доступно по ссылке выше. Если хотите запустить локально:

1. Установите зависимости:
   pip install -r requirements.txt

2. Запустите приложение:
   python app.py

3. Откройте в браузере:
   http://127.0.0.1:5000
