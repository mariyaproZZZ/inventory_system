# Система учёта складских запасов

Веб-приложение для учёта товаров на складе: добавление, редактирование количества/цены и удаление.

---

## 🚀 Демо

👉 Запустите локально: `http://127.0.0.1:5000` 👈

---

## 📋 Основные возможности

| Функция | Описание |
|---------|----------|
| ➕ Добавление | Добавление товара с названием, количеством и ценой |
| ✏️ Редактирование | Изменение количества и цены любого товара |
| 🗑️ Удаление | Удаление товара из списка |
| 📊 Просмотр | Таблица со всеми товарами на складе |
| 💾 Автосохранение | Все изменения сразу сохраняются в базе данных |

---

## 🛠️ Технологический стек

| Компонент | Технология |
|-----------|------------|
| Серверная часть | Python + Flask |
| База данных | SQLite |
| Интерфейс | HTML5, CSS3 |
